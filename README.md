# Primeros Auxilios

## Descripción
Este proyecto consiste en una pagina web  en HTML sobre "primeros auxilios". El objetivo de esta pagina es brindar informacion clara y accesible sobre como actuar en situaciones de emergencias, como accidentes, heridas, quemaduras o desmayos.

## Contenido:
- Imagen
- Menu principal
- Informacion sobre primeros auxilios
- Guia sobre como actuar en caso de emergencia
- Tipos de Emergencia
- Tecnicas escenciales
- Estadisticas
- Contacto
